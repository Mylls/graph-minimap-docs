# Support

If you need help with Graph Minimap:

- Check the [Quick Start](quick-start.md) page
- Review the [Settings](settings.md) page
- Review [Troubleshooting](troubleshooting.md)
- Use the support or contact information provided on the Fab product page

## Good Information To Include In A Support Request

When reporting an issue, include:

- Unreal Engine version
- whether the plugin was installed into a fresh project or an existing one
- graph type involved (Blueprint, Material, Behavior Tree, or Niagara)
- what you expected to happen
- what actually happened
- whether the issue occurs when opening the minimap from the `Window` menu
