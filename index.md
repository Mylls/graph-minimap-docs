# Graph Minimap

A minimap for Unreal graphs that makes large Blueprints, Materials, Behavior Trees, and Niagara systems easier to read and navigate.

Graph Minimap is an Unreal Engine editor utility that helps you understand graph structure faster, move around large graphs more easily, and jump directly to the area you need.

## What It Does

- Adds a dedicated minimap window to supported Unreal graph editors
- Lets you pan the active graph by clicking and dragging inside the minimap
- Supports mouse-wheel zoom directly from the minimap
- Includes search for nodes and comments
- Supports bookmarks for frequently used graph locations
- Includes dense graph handling with a focus lens for local detail
- Remembers minimap size and placement between uses

## Supported Graph Types

- Blueprint graphs
- Material graphs
- Behavior Tree graphs
- Niagara graphs
