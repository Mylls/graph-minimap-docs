# Graph Support

Graph Minimap supports the following Unreal graph types.

## Blueprint

Blueprint graphs are fully supported and are the primary workflow target for the plugin.

## Material

Material graphs are supported and can be navigated through the minimap like other supported editors.

## Behavior Tree

Behavior Tree graphs are supported for overview and navigation workflows.

## Niagara

Niagara graphs are supported.

If a keyboard shortcut does not trigger in Niagara, use the graph editor `Window` menu to open the minimap. This is usually caused by editor shortcut conflicts rather than a plugin installation issue.
