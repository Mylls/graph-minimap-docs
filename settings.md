# Settings

Open Graph Minimap settings here:

`Editor Preferences > Plugins > Graph Minimap`

## Graph Minimap

### Show Reroute Nodes
Controls whether reroute nodes are shown in the minimap.

### Minimum Node Size
Sets the minimum visible size for nodes in the minimap.

### Save Minimap Location & Size
Remembers the minimap's size and placement between uses.

### Auto-open Minimap In New Blueprints
If enabled, opening the minimap once allows newly opened Blueprint graphs to auto-open it until you manually close it.

## Rendering

### Enable Dense Graph Mode
Switches to dense graph behavior for very large graphs.

### Dense Graph Mode Node Threshold
Controls when dense graph mode activates.

### Show Comment Tags
If enabled, compact comment labels can appear when space allows.

### Focus Lens Opacity
Controls the opacity of the focus lens background.

### Focus Lens Scale
Controls the size of the focus lens. Setting it to 0 hides the focus lens entirely.

### Show Selected Node Links
If enabled, the minimap can draw simplified links for selected nodes.

### Color Profile
Choose between Normal, Deuteranopia, Protanopia, and Tritanopia color profiles.

## Bookmarks

### Bookmark Slots
Controls how many bookmark slots are available in the bookmarks dropdown.
