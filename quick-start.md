# Quick Start

## Open the Minimap

In a supported graph editor:

1. Open the graph you want to work in.
2. Use the `Window` menu.
3. Choose `Graph Minimap`.

A keyboard shortcut can also toggle the minimap. If it does not trigger in a specific editor, check Unreal Editor keyboard shortcut conflicts and rebind it if needed.

## Basic Controls

- Drag inside the minimap to pan the main graph view
- Use the mouse wheel over the minimap to zoom the graph view
- Use the search button to search nodes and comments
- Use bookmarks to save and return to important graph locations
- Use the settings button to open the plugin settings page directly

## Large Graphs

Graph Minimap stays useful on both small and large graphs.

For dense graphs:

- the minimap can switch into dense graph mode
- the focus lens helps show local detail
- the viewport box still shows where you are in the larger graph

## Recommended First Settings Check

Open:

`Editor Preferences > Plugins > Graph Minimap`

Recommended defaults for new users:

- Show Reroute Nodes: Off
- Minimum Node Size: 6
- Enable Dense Graph Mode: On
- Dense Graph Mode Node Threshold: 200
- Show Comment Tags: Off
- Focus Lens Opacity: 1.0
- Focus Lens Scale: 0.75
- Show Selected Node Links: Off
- Auto-open Minimap In New Blueprints: On
